# 🦭 Seal Bounce 

Um jogo 2D casual e divertido onde você controla uma foca simpática que precisa equilibrar e fazer quicar os objetos que caem do céu! Teste seus reflexos e tente conseguir a maior pontuação sem deixar nada cair na água.

---

## 🕹️ Como Jogar

O objetivo é simples: mantenha os objetos no ar o maior tempo possível!

* ⬅️ **Seta para a Esquerda / A:** Move a foca para a esquerda.
* ➡️ **Seta para a Direita / D:** Move a foca para a direita.
* 🪃 **Mecânica:** Posicione a foca embaixo dos objetos que caem do céu para fazê-los quicar de volta. Cada quique bem-sucedido garante pontos!

---

## ✨ Funcionalidades

* **Controles Responsivos:** Movimentação fluida para os lados.
* **Física de Quique:** Os objetos reagem de forma dinâmica ao bater na cabeça da foca.
* **Sistema de Pontuação:** Descubra quantos quiques você consegue fazer antes de falhar.
* **Dificuldade Progressiva:** Conforme o tempo passa, mais objetos caem ou a velocidade aumenta! (Opcional: ajuste conforme o seu jogo).

---

## 🛠️ Tecnologias Utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

* **Linguagem:** [Odin](https://odin-lang.org/) 
* **Engine de Física:** [Box2D](https://box2d.org/) (via pacote nativo `vendor:box2d`)
* **Renderização Gráfica:** [Raylib](https://www.raylib.com/) (via pacote nativo `vendor:raylib`)
